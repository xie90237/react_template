# 这是一个react简易的 全面的 便捷的模板
> 其中包含了下述框架 并都有相关示例
- vite
- react18 + ts
- react-router
- axios
- react-i18next
- Zustand
- styled-components
- antd

> 我的开发环境为
> node: v18.16.0
> 包管理工具: yarn
> 系统: win10
> 我的开发环境是这个 防止 使用别的版本之类的 产生各种各样的问题 我主要也懒得改 给一个参考
